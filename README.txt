・はじめに
　「お前は今まで貯めた硬貨の枚数をおぼえているのか―――？」

　MyBankは非常にシンプルな硬貨貯金箱アプリです。
　日本円の硬貨6種類の貯金を管理することが出来ます。それぞれの硬貨を999枚まで貯金出来ます。

　◇硬貨を左クリック　：硬貨を1枚貯金
　◇硬貨を右クリック　：硬貨を1枚引出
　◇ブタ貯金箱をクリック　：貯金状態を保存

　MyBank -> bin -> Release -> MyBank.exe　のショートカットをデスクトップに置くなどして使用してください。
